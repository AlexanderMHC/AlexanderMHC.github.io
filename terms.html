HTML Terms

  • <a> and </a>: Defines a hyperlink, which is used to link from one page to another
  • <article> and </article>: Specifies independent, self-contained content.
  • <aside> and </aside>: Defines some content aside from the content it is placed in.
  • <b> and </b>: Specifies bold text without any extra importance.
  • <body> and </body>: Defines the document's body.
  • <br> and </br>: Inserts a single line break.
  • <em> and </em>: Content inside is typically displayed in italic.
  • <footer> and </footer>: Defines a footer for a document or section.
  • <head> and </head>: A container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
  • <header> and </header>: Represents a container for introductory content or a set of navigational links.
  • <hr> and </hr>: Displays a horizontal rule that is used to separate content (or define a change) in an HTML page.
  • <img> and </img>: Used to embed an image in an HTML page.
  • <li> and </li>: Used inside ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).
  • <meta> and </meta>: Always goes inside the <head> element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.
  • <nav> and </nav>: Defines a set of navigation links.
  • <ol> and </ol>: Defines an ordered list. An ordered list can be numerical or alphabetical.
  • <p> and </p>: Defines a paragraph.
  • <q> and </q>: Defines a short quotation (<blockquote> are used for long quotations).
  • <section> and </section>: Defines a section in a document.
  • <title> and </title>: Defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

CSS Terms

  • display property: specifies the display behavior (the type of rendering box) of an element.
  • color property: specifies the color of text.
  • background-color property: sets the background color of an element.
  • height property: sets the height of an element.
  • width property: sets the width of an element.
  • margin property: sets the margins for an element, and is a shorthand property for the following properties.
  • font-family property: specifies the font for an element.
  • padding property: sets the space between its content and its border.
  • text-shadow property: adds shadow to text.
  • text-align property: specifies the horizontal alignment of text in an element.
